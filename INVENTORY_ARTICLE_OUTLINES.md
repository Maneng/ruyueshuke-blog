## 第一篇：库存管理系统全景图

### 标题
库存管理系统全景图：从安全库存到呆滞处理的完整体系

### 概要
系统化介绍库存管理的核心概念、经典模型、系统架构和实战方法。

### 大纲结构

#### 1. 库存管理概述
- 什么是库存管理
  - 定义：在保证供应的前提下，最小化库存成本
  - 核心目标：降低库存成本 + 提高服务水平
- 库存的三大成本
  - 持有成本：仓储费、资金占用、损耗
  - 订货成本：订单处理、运输、验收
  - 缺货成本：销售损失、客户流失、紧急采购
- 库存的价值
  - 缓冲需求波动
  - 应对供应不确定性
  - 规模经济（批量采购降低单价）

#### 2. 库存分类管理
- ABC分类法
  - A类：20%商品占80%销售额（重点管理）
  - B类：30%商品占15%销售额（正常管理）
  - C类：50%商品占5%销售额（简单管理）
- XYZ分类法
  - X类：需求稳定（预测准确率高）
  - Y类：需求波动（预测准确率中等）
  - Z类：需求不规律（预测准确率低）
- 组合分类
  - AX类：高价值+稳定需求（精细化管理）
  - CZ类：低价值+不规律需求（粗放式管理）

#### 3. 经典库存模型
- EOQ模型（经济订货批量）
  - 公式：EOQ = √(2DS/H)
  - D：年需求量，S：单次订货成本，H：单位持有成本
  - 适用场景：需求稳定、补货及时
- ROP模型（再订货点）
  - 公式：ROP = 日均需求 × 提前期 + 安全库存
  - 适用场景：固定提前期、需求稳定
- 安全库存计算
  - 公式：SS = Z × σ × √L
  - Z：服务水平系数（95%→1.65，99%→2.33）
  - σ：需求标准差，L：提前期

#### 4. 库存控制策略
- 定量订货法（Q系统）
  - 当库存降至ROP时，订购固定数量Q
  - 适用：单价高、需求稳定
- 定期订货法（P系统）
  - 每隔固定周期T，订购至目标库存S
  - 适用：品类多、需求波动大
- JIT准时制
  - 零库存理念
  - 适用：汽车制造等

#### 5. 库存数据模型
- 库存主数据
  - SKU、仓库、库存数量、安全库存
- 库存事务
  - 入库、出库、盘点、调拨

#### 6. 行业案例
- 沃尔玛的库存管理
  - VMI（供应商管理库存）
  - 自动补货系统
  - RFID实时追踪
- 亚马逊的库存优化
  - 预测性调拨
  - 动态安全库存
  - 智能补货

#### 7. 库存管理挑战
- 需求预测不准
- 供应链不确定性
- 库存数据不准确
- 呆滞库存积压

#### 8. 未来趋势
- AI需求预测
- 智能补货
- 区块链溯源
- 数字孪生

## 第二篇：安全库存计算与动态调整

### 标题
安全库存计算：从经典公式到动态调整的实战方法

### 概要
深入讲解安全库存的计算方法和动态调整策略。

### 大纲结构

#### 1. 安全库存的作用
- 应对需求波动
- 应对供应不确定性
- 提高服务水平

#### 2. 经典计算公式
- 基础公式：SS = Z × σ × √L
- 参数详解
  - Z：服务水平系数（查正态分布表）
  - σ：需求标准差（历史数据计算）
  - L：提前期（供应商交期）
- 实例计算
  - 假设：日均需求100，标准差20，提前期7天，服务水平95%
  - SS = 1.65 × 20 × √7 = 87件

#### 3. 动态调整策略
- 按销售季节调整
  - 旺季提高安全库存
  - 淡季降低安全库存
- 按促销活动调整
  - 大促前提前备货
  - 大促后快速消化
- 按供应商稳定性调整
  - 稳定供应商：降低安全库存
  - 不稳定供应商：提高安全库存

#### 4. Java代码实现
```java
public class SafetyStockCalculator {
    // 计算安全库存
    public double calculate(double z, double sigma, double leadTime) {
        return z * sigma * Math.sqrt(leadTime);
    }

    // 根据服务水平获取Z值
    public double getZValue(double serviceLevel) {
        if (serviceLevel >= 0.99) return 2.33;
        if (serviceLevel >= 0.95) return 1.65;
        if (serviceLevel >= 0.90) return 1.28;
        return 1.0;
    }
}
```

- 京东的安全库存
  - 多仓协同
  - 动态调整
  - 大数据预测

## 第六篇：需求预测算法与实现

### 标题
需求预测算法：从时间序列到机器学习的实战

### 概要
详细讲解需求预测的经典算法和工程实现。

### 大纲结构

#### 1. 时间序列预测
- 移动平均法
  - 简单移动平均（SMA）
  - 加权移动平均（WMA）
- 指数平滑法
  - 一次指数平滑（适用于无趋势）
  - 二次指数平滑（适用于有趋势）
  - 三次指数平滑（适用于有季节性）
- ARIMA模型
  - 自回归（AR）
  - 差分（I）
  - 移动平均（MA）

#### 2. 机器学习预测
- 线性回归
  - 多元线性回归
  - 岭回归（Ridge）
- 决策树/随机森林
  - 特征工程
  - 模型训练
- LSTM神经网络
  - 适用于长序列预测
  - 捕捉长期依赖

#### 3. 预测准确率评估
- MAPE（平均绝对百分比误差）
  - MAPE = Σ|实际值-预测值|/实际值 / n
  - 行业基准：< 20%合格，< 10%优秀
- MAD（平均绝对偏差）
- RMSE（均方根误差）

#### 4. Java代码实现
```java
public class DemandForecast {
    // 移动平均法
    public double movingAverage(List<Double> history, int period) {
        return history.stream()
            .skip(history.size() - period)
            .mapToDouble(Double::doubleValue)
            .average()
            .orElse(0.0);
    }

    // 指数平滑法
    public double exponentialSmoothing(List<Double> history, double alpha) {
        double forecast = history.get(0);
        for (int i = 1; i < history.size(); i++) {
            forecast = alpha * history.get(i) + (1 - alpha) * forecast;
        }
        return forecast;
    }
}
```
